## Getting Started
```bash
# 1. Clone the Repository
git clone https://github.com/lusung33/AAAI26_LoL_MinimapDetection.git
cd AAAI26_LoL_MinimapDetection

# 2. Environment Setup
conda env create -f environment.yml
conda activate esports312

# 3. Download Datasets and Sample Weights
# You can use HF Link to download
# Replay Dataset: ./data/
# Sample Model: ./results/model/

# 4. Generate Official Riot API Key
# Please read ./src/key/README.md
```

## Synthetic Data Generate
```bash
python -m scripts.data.synthetic_data_generator_v2 # set params (n_train, n_val, n_test)
```

## Train Model
```bash
# pretrain model
python -m scripts.train.model_train --file_name <your_synthetic_images_config> --epoch 100 --batches 32 --imgsz 256 # set params

# finetune model
python -m scripts.train.model_finetune --model <pretrained model name> # set params
```

## Visualize
```bash
# download models before run
python -m scripts.vis.visualizer --video_name vis_sample_video.mp4 --game_id KR_7691650081 --save --nfps 1
```
## Evaluate
```bash
# download models before run
python -m scripts.eval.evaluator
```
